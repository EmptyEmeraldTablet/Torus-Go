# 环面围棋（单棋盘版）

在线示例：https://torus-gomoku.pages.dev/

**游戏规则**
- 棋盘为环面拓扑：上下、左右边界相连。
- 棋子落在网格交叉点上，黑先白后轮流落子。
- 提子规则与气的判断沿环面相邻关系计算。
- 规则集对齐 Chinese：面积计分、简单劫（Simple Ko）、不允许自杀、贴目 7.5。
- 双方连续停一手后进入清理阶段，可点击棋子标记死子，按面积规则结算（含贴目）。
- 支持拖动平移视图，查看环面位置关系。

**当前开发情况**
- 已实现：棋盘渲染、环面坐标转换、提子与劫判断、面积计分、拖动平移、停一手、重置、撤回、棋盘大小切换、状态栏显示、人机对弈（简单 AI）。
- 待完善：规则配置（超劫/地盘/自杀）、移动端触控体验细化、性能与交互细节优化、自动测试。

**本地开发**
- 依赖：Node.js + npm
- 启动开发环境：
  - `npm install`
  - `npm run dev`

**人机对弈**
- 选择“对弈模式”为“人机对弈”。
- 选择 AI 执黑或执白，系统会自动在 AI 回合落子。
